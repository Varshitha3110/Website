An AI Hand Detector uses computer vision to locate and track human hands in real-time video. It identifies key hand landmarks like fingertips and joints, enabling applications like gesture recognition, virtual input systems, and interactive interfaces without physical touch.


üîç How Does It Work?

  1. Video Input: Captures frames from a webcam or camera feed.
  2. Hand Landmark Detection: MediaPipe processes each frame to detect and track 21 hand landmarks.
  3. Bounding Box & Labeling: Draws a box and labels the hand (e.g., Left/Right) based on coordinates.
  4. Finger State Analysis: You can calculate which fingers are raised, angles between joints, etc.
